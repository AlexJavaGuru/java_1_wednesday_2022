Проверьте все ваши классы и уберите с них идентификатор доступа public.

В Java можно создать классы двумя способами:
- с указанием ключевого слова public,
- без указания ключевого слова public.

При создании класса среда разработки автоматически
добавляет ключевое слово public в объявление класса.

Ключевое слово public является идентификатором доступа.
Если при объявлении класса использовалось ключевое слово public
то это означает, что созданный вами класс можно использовать
в любом другом месте проекта. Это означает, что созданный
вами класс могут заимпортировать и использовать другие
слушатели курса. Обычно это может произойти по ошибке,
так как у всех классы называются одинаково.

Для того, что бы предотвратить использование ваших классов
другими слушателями курса по ошибке удалите ключевое слово public
в объявлении ваших классов.

До:
public class Robot {

}

После:
class Robot {

}

Если ваш класс будет объявлен без ключевого слова public это означает,
что его можно будет использовать только в том пакете, в котором этот
класс объявлен. Это предотвратит использование ваших классов другими
слушателями курса по ошибке.

Впредь в течении курса создавайте классы без ключевого слова public.
Применяйте ключевое слово public только тогда, когда вы хотите сделать
ваши классы доступными для всех.
