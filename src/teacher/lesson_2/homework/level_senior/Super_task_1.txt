При написании Java кода для названий классов принято использовать
нотацию под названием CamelCase (с англ. — «ВерблюжийРегистр»,
также «ГорбатыйРегистр», «СтильВерблюда») — стиль написания
составных слов, при котором несколько слов пишутся слитно без пробелов,
при этом каждое слово внутри фразы пишется с прописной буквы.
Стиль получил название CamelCase, поскольку прописные буквы
внутри слова напоминают горбы верблюда (англ. Camel).

Так же в Java принято в названии класса первую букву
делать всегда заглавной. Компилятор и синтаксис языка Java
это не требуют, но чёткое следование этому правилу сильно облегчает
чтение программ.
