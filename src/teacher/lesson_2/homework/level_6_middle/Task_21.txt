При программировании более удобно и быстро выполнять все
действия с помощью клавиатуры, не переключаясь на мышку.

Научитесь комитить код в Git без мышки!

Alt (слева) + 1 - открывает окно Project в котором видна структура проекта.

Стрелкой вверх перемещаемся в самый корень проекта.

Вызывает всплывающее меню используя клавиатуру.

Стрелочки: вверх, вниз, влево, вправо - помогают выбирать
нужный пункт меню.

Git -> Commit Directory ...

Пишем сообщение для комита.

Ctrl (слева) + Enter - это вызов подсвеченной синим кнопки
в любом диалоговом окне (в нашем случае Commit).


После Commit проделайте тоже самое с командой Push:

Alt (слева) + 1 - открывает окно Project в котором видна структура проекта.

Стрелкой вверх перемещаемся в самый корень проекта.

Вызывает всплывающее меню используя клавиатуру.

Стрелочки: вверх, вниз, влево, вправо - помогают выбирать
нужный пункт меню.

Git -> Repository -> Push ...
