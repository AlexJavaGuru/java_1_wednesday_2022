Проверьте все названия созданных вами классов.

В Java принято называть классы с заглавной буквы.
Хотя компилятор и даёт вам возможность объявить класс с маленькой буквы,
это сильно ухудшает читаемость Java кода особенно в больших приложениях,
которые вы вскоре будете писать самостоятельно. Поэтому сразу приучаемся
называть классы правильно!

Если вы обнаружили класс названный с маленькой буквы, то исправьте его название.

Alt (слева) + 1 - открываем окно проекта.

Стрелками вверх, вниз, вправо, влево выбираем класс
название которого вы хотите исправить.

Вызывает всплывающее меню используя клавиатуру.

Refactor -> Rename...

И даём новое название вашему классу.

Быстрая комбинация клавиш, когда курсов находится на имени класса в редакторе кода: Shift (слева) + F6.
