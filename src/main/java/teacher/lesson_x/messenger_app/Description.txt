Messenger App

Цель: создать p2p (Pear to Pear) мессенжер приложение.

Главная задача приложения: дать возможность пользователям
напрямую посылать сообщения друг другу с разных компьютеров,
тоесть по сети.

В минимальной версии приложения должно быть:
- консольный UI;
- при старте программы создаётся одна чат комната
  в рамках которой и будет проходить все общение;
- возможность послать текстовое сообщение на указанный IP адрес;
- приложение должно принимать входящие сообщения на определённом
  порту и показывать их содержимое в чате.

После того, как заработает минимальная версия приложения в
программу можно добавить следующую функциональность:
- возможность давать пользователям ники, тоесть при старте
  приложения пользователь указывает свой ник и потом все
  сообщения от этого пользователя показываются с ником этого
  пользователя;
- сохранение истории чата в файл на диске;
- поиск сообщений в истории чата;
- возможность послать приватное сообщение выбранному пользователю в чате;
- возможность создания новых комнат для общения в чате;
- возможность послать приглашение участнику чата присоединиться в комнату;
- красивый UI
- и так далее.
