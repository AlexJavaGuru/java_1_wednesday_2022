Вводим новую функциональность: анализ расходов.

Банк, для которого вы пишите этот софт, решил проанализировать
на что тратят деньги его клиенты. Банк хочет узнать какие
категории расходов есть у клиента, сколько денег он тратит
на каждую категорию своих расходов и сколько это составляет
в процентном отношении от всех затрат владельца банковского аккаунта.

Банк планирует внедрить анализ расходов в процессы выдачи кредитов своим клиентам.

На данном этапе банк интересуют следующие категории расходов:
- комунальные платежи,
- еда,
- развлечения,
- азартные игры,
- кредиты,
- всё остальное.

Для описания категорий расходов создадим enum ExpenseCategory:

enum ExpenseCategory {

    COMMUNAL_PAYMENTS,  // комунальные платежи
    FOOD,               // еда
    ENTERTAINMENT,      // развоечения
    GAMBLING,           // азартные игры
    LOANS,              // кредиты
    OTHERS              // остальное

}
