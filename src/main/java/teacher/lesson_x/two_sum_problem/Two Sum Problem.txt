Дан массив целых чисел и отдельно целое число, которое является целью.

Верните индексы двух чисел из массива, при сложении которых получается число равное цели.

Вы можете ожидать, что входные данные имеют только одно правильное решение
и вы не можете использовать один и тот же елемент массива дважды.

Вы можете возвращать результат в любом порядке.

Пример:

Input: array = [1, 2, 5, 8]  target = 7
Output: [1, 2]  или [2, 1]
array[1] + array[2] = 2 + 5 = 7
