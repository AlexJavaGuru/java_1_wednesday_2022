Игра в теннис имеет довольно не привычную для новичка
систему подсчёта очков. Вас наняли для написания компьютерной программы,
которая должна подсчитывать очки в тенисной игре.

Далее перечислены правила тениса, которые вам придётся учитывать
при разработке программы:

- Гейм в тенисе выигрывает тот игрок, кто первым наберёт
  минимум четыре очка, при этом у него должно быть на два
  очка больше чем у противника.
- Очки в тенисном гейме описываются следующим образом:
  очки от нуля до трёх - "Love", "Fifteen", "Thirty", and "Forty"
  соответственно.
- Если оба игрока набрали хотя бы по три очка каждый
  и у обоих игроков одинаковое количество очков,
  то результат "Deuce".
- Если оба игрока набрали хотя бы по три очка каждый
  и один из игроков набрал на одно очко больше другого,
  то результат игры "Advantage" в пользу игрока с наибольшим
  количеством очков.

Ваша программа должна подсчитывать очки только
для одного гейма игры. Сеты и матчи не входят в
контекст этой задачи.

Ваша задача создать класс, который будет
реализовать интерфейс TennisGame:

interface TennisGame {

    void wonPoint(String playerName);

    String score();

}

Когда игрок выигрывает очко, у вашего класса будут
вызывать метод win() и передавать в качестве
параметра имя игрока выигравшего очко.
Потом у вашего класса будет вызываться метод
score(), который должен вернуть строку с информацией
о текущем счёте игры. Полученную таким образом информацию
предпологается выводить на табло.

Пример:

TennisGame game = new TennisGame("player1", "player2");

game.score(); -> "Love - Love"
game.wonPoint("player1");
game.score(); -> "Fifteen - Love"
game.wonPoint("player1");
game.score(); -> "Thirty - Love"
game.wonPoint("player1");
game.score(); -> "Forty - Love"
game.wonPoint("player2");
game.score(); -> "Forty - Fifteen"
game.wonPoint("player2");
game.score(); -> "Forty - Thirty"
game.wonPoint("player2");
game.score(); -> "Deuce"
game.wonPoint("player2");
game.score(); -> "Advantage player2"
game.wonPoint("player2");
game.score(); -> "Win player2"

Создайте реализацию интерфейса TennisGame и напишите
автоматические тесты.
