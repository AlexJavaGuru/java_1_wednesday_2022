Знакомимся с дизайн паттернами.

Найти в интернете информацию о дизайн паттерне Builder.

Примеры:
https://dzone.com/articles/design-patterns-the-builder-pattern
https://howtodoinjava.com/design-patterns/creational/builder-pattern-in-java/

Создайте Builder класс для одной из моделей данных из нашего курса и продемонстрируйте его использование.

Протестируйте созданный вами Builder! :)

Подсказка на будущее: дизайн паттерна Builder можно использовать для создания
сложных объектов с данными (у которых много свойств) в тестах. Это сделает
ваши тесты более читабельными.
