Подключите к среде разработки IntelliJ Idea
установленный на своём компьютер Git.

Для подключения Git к среде разработки IntelliJ Idea нужно:
- открыть настрой среды разработки: File -> Settings ...
- в открывшемся диалоговом окне в верхнем левом углу
  в поле поиска ввести "Git" и нажать Enter
- убедиться, что слева под полем поиска выбрана закладка
  Version Control -> Git
- в основной части окна справа на самом верху найти
  Path to Git executable: и в этом поле указать путь до файла git.exe
  (этот файл находится в той папке куда вы проинсталировали Git/bin/git.exe)
- справа от поля Path to Git executable: расположена кнопка Test,
  после указания пути к файлу git.exe нажмите её и если вы сделали
  все верно то под полем появится надпись с версией Git.
