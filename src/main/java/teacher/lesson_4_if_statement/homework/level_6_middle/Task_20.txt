Проверьте весь ваш код и убедитесь, что для каждого тела if оператора вы используете
правильные отступы.

Правильное форматирование кода сильно облегчает чтение программы.

Пример без отступов:
        currentPrice = newPrice;
        if (newPrice < minPrice) {
        minPrice = newPrice;
        }
        if (newPrice > maxPrice) {
        maxPrice = newPrice;
        }

Пример с не правильными отступами:
        currentPrice = newPrice;
        if (newPrice < minPrice) {
        minPrice = newPrice;      // в одном месте нет отступов
        }
        if (newPrice > maxPrice) {
            maxPrice = newPrice;  // в другом есть
        }


Пример с правильными отступами:
        currentPrice = newPrice;
        if (newPrice < minPrice) {
            minPrice = newPrice;
        }
        if (newPrice > maxPrice) {
            maxPrice = newPrice;
        }


Код с правильным форматингом (отступами) читается и понимается намного проще.

Не забывайте в своих программах использовать отступы при написании тела if оператора!
