Проверьте весь ваш код и убедитесь, что у каждого if оператора есть фигурные скобки {}
ограничивающие его тело.

Хотя фигурные скодки {} при объявлении оператора if являются не обязательным элементом,
указывать их в своей программе очень желательно. Фигурные скобки очень сильно облегчают
чтение программы, при их использовании сразу видно какая часть кода относится к телу
if оператора, а какая нет.

Пример без скодок:
        currentPrice = newPrice;
        if (newPrice < minPrice)
            minPrice = newPrice;
         else if (newPrice > maxPrice)
            maxPrice = newPrice;
укп
Пример со скобками:
        currentPrice = newPrice;
        if (newPrice < minPrice) {
            minPrice = newPrice;
        } else if (newPrice > maxPrice) {
            maxPrice = newPrice;
        }

Код со скобками читается намного проще, при его чтении не надо задумываться о том,
какая часть кода относится к if оператору.
